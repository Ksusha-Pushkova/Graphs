
# Graph Visualizer

## Функциональность

- ✅ Добавление/удаление вершин и ребер
- ✅ Поддержка ориентированных и неориентированных графов
- ✅ Визуализация графа с весами ребер
- ✅ Алгоритмы обхода: DFS и BFS
- ✅ Алгоритм Дейкстры для поиска кратчайшего пути
- ✅ Красивый розовый интерфейс

##  Запуск проекта

```bash
# Компиляция
javac GraphGUI.java

# Запуск
java GraphGUI
